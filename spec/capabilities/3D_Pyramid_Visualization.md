# 3D Pyramid Visualization

Users can inspect the repository knowledge graph as a 3D pyramid. Vision nodes sit near the top, code nodes sit at the bottom, and cross-cutting notes are placed around the main stack.

The pyramid should preserve abstraction density: high-level layers stay sparse and readable, while denser lower layers are revealed through progressive exploration instead of being shown all at once by default.
